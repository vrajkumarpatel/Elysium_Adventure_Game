# Module 2

Exercises and practice programs for CSS225 Module 2.

## Contents
- Python scripts: BMI calculator, Morse code, greetings, sums/products.
- Reference document: `Module 2 Lab Activity.docx`.

## Run Scripts
- Ensure Python 3.x is installed.
- From this folder, run any script with: `python <script_name>.py`.

## Notes
- Try `BMI.py`, `Morse_Code.py`, and `Sum_Product.py` as starters.
